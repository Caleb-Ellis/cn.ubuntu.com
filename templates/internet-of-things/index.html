Ubuntu Core on the Internet of Things ================ #}

物联网世界的 Ubuntu Core

Snappy Ubuntu Core 使稳固的安全性、可靠的更新和庞大的 Ubuntu 生态系统变得触手可及，并将深受开发人员喜爱的云平台带到各种物联网、联网设备和自主设备。

Ubuntu Core是一套安全、轻量级、应用激活且具有事务化更新的Ubuntu。 “snappy”英文原意可以理解为简洁轻快，而“snappy”应用在这里是指一款全新更加简便的应用程序打包系统。

现在，众多 32 位和 64 位 ARM 和 X86 平台上均可运行 snappy Ubuntu Core。

{# ============== 2. Secure critical infrastructure and devices ============= #}

保护关键基础架构和设备

关键基础架构系统是间谍和破坏活动的一种攻击媒介。Ubuntu Core 设备会自动进行更新，快速修补系统漏洞。Ubuntu Core 还提供一流的基于内核容器的应用程序隔离功能，尽量减少第三方应用程序中的错误和漏洞的影响。

经过认证且受支持的 Ubuntu Core 平台为联网设备的安全和防护树立了全新的标准。在认证平台上选择 Ubuntu Core 的设备制造商现在有了一个非常受欢迎的平台，该平台可以满足企业和政府对于安全更新和管理的要求。

更聪明、更安全且令人惊叹的自主智能设备

早期在联网设备上使用 Ubuntu 的用户是活跃于机器人、无人机和开放式物联网中心领域的开拓者。Ubuntu Core 是超级智能设备的安全平台，它提供一个应用商店，可将最新的软件直接推送到您的设备，并轻松与每个云建立连接。

无人机和其他智能设备的商业供应商现在可以通过一个开放的平台自动提供可靠的软件更新，以及向客户销售软件。

“我们很高兴地宣布 Erle-Copter 的诞生。它是世界首款由 Ubuntu Core 提供支持的无人机，能够自动确保安全，并可从应用商店升级其他附加功能。开放的平台会吸引创新者和专家开展合作与竞争，我们很激动能够引领开源无人机的教育、研究和发明之路。”

Erle Robotics首席技术官， Victor Mayoral Vilches

{# ============ 3. Software-defined appliances simplify telco CPE =========== #}

利用软件定义的设备简化电信行业的 CPE

Canonical 正在与电信运营商开展合作，以 snappy Ubuntu Core 内置的运营商级系统和应用程序更新机制为基础，简化用户端设备 (Customer Premises Equipment, CPE) 的采购、部署和维护。对于众多以 Snappy 应用程序和框架形式交付的软件定义 CPE 设备而言，用户可以期待在普通商用 X86 和 ARM 硬件上构建标准管理系统，而不必选择管理系统分散的品牌硬件设备。

更智能的交换机、不同凡响的路由器

新一代交换机能够运行 Ubuntu Core 和一系列以网络为中心的应用程序。

Snappy Ubuntu Core 15.04 现在可作为白牌交换机和 OCP 交换机设计的开发用 Linux 内核。得益于此，任何 ODM、网络操作系统供应商或白牌交换机经销商提供的交换机，都可以带有与客户在其服务器和 OpenStack 环境中所用相同的主流 Linux 发行版。

进一步了解适用于白牌交换机的 Snappy

{# ================= 4. Enterprise grade, industry hardened ================= #}

企业级性能在工业领域的再一次提升

Ubuntu 在企业数据中心的广泛部署为其进入工业控制和管理系统奠定了基础。

获取出色的企业软件

随着智能设备进入企业，人们需要将它们与现有的身份识别、管理和监控系统相集成，而 Ubuntu 在这方面提供了极其广泛的云时代解决方案。例如，Forgerock 是一个在身份及访问权限管理领域被广泛采用的领先平台，能够管理包括人员、物品、场所和活动在内的所有要素。现在，该平台将面向 Ubuntu Core 系统的开发人员提供其解决方案。

{# ========== 5. A marketplace for incredible apps on open devices ========== #}

面向开放设备的精彩应用市场

通用的可扩展设备，如集线器、机顶盒和网关等，为全球开发人员社区开发的各类应用开辟了一个庞大的应用市场。Ubuntu 让人们可以轻松在笔记本电脑上为各种新奇设备开发精彩的应用，在云中测试这些应用，然后直接将它们发布到涵盖各种设备的全球市场。

开发人员的最爱

Ubuntu 平台拥有大量软件帮助开发人员进行复杂的处理工作，例如视觉处理、传感器处理以及运动和位置数据的处理。Ubuntu在开发人员中颇受欢迎，因此经常能在无人驾驶汽车、娱乐控制系统、深空任务控制中心和智能显示系统等诸多项目中看到“Ubuntu 的实战应用”。Ubuntu Core 为那些面向全球推出、很难以物理方式实际访问、连接到互联网获取更新和安全修补程序的产品提供了一个随时可以投入生产的平台。

与 Azure 和 AWS 建立云连接

微软和亚马逊已同意在 Ubuntu Core 上为开发人员发布其物联网开发人员 API。

“智能工业系统需要安全的云后端来存储和分析数据。微软和 Canonical 正在开展合作，为 开发人员提供开发人员 API 来支持Ubuntu Core。这次合作将简化云支持的设备开发。” 微软的技术研究员 John Shewchuk 说。

从云到设备，只需一个平台

Ubuntu Core 从云到设备均采用同一个平台。为嵌入式设备进行开发从未如此简单。云中的 Ubuntu Core 是一个理想的测试和开发平台。你可以在线模拟你的设备，并可根据需要启动数千个模拟设备。 Canonical 确保云和设备中的 Ubuntu Core 具有完全相同的 API，并能接收完全相同的安全更新。

{# ====================== 6. Range of supported devices ===================== #}

支持极其广泛的设备

同时支持 ARMv7 和 X86-64。使用标准 PC 设备使设备的原型开发变得简单轻松。一台空闲的笔记本电脑或 PC 就能作为你最初的开发板。你可以将一台旧 PC 变成一个家庭存储服务器，或是在空闲的笔记本电脑或虚拟机上开发机器人应用的原型。

从低成本的消费级开发板…

从 35 美元的 ARMv7 Odroid-C1（提供 1 GB RAM、千兆网络连接和众多 RaspberryPi 式的扩展端口和引脚）到 179 美元的 2 GHz Octacore Odroid-XU3（具有 2 GB RAM 和多种显示及 I/0 支持），Ubuntu 适用于各种 ARM 开发板。由于知道可以在制造时选择合适的芯片，您可以胸有成竹地开发应用程序。缩短从概念创新到产品上市的时间，是基于 Ubuntu 的设备的一项关键优势。

Ubuntu Core 需要具有 128 MB RAM 的 600 MHZ 处理器。设备需要 4 GB 闪存用于恢复出厂设置和系统回滚。Ubuntu Core 本身将使用 40 MB RAM，其余 RAM 留给应用程序使用。建议使用 Beaglebone Black、Odroid-C1 和 Raspberry Pi 2 作为 ARM 开发板。对于 x86，任何空闲的笔记本电脑、PC 或虚拟机都可以作为开发平台。

… 到高端服务器设备

Ubuntu Core 同样可为速度最快的 Xeon 服务器和大规模多核心 ARM CPU（如 Cavium 的迅雷）提供良好支持。从为新一代交换机和路由器提供支持的高端服务器设备，到面向容器的超大规模基础架构和以机架为中心的转码引擎，Ubuntu Core 为设备设计人员和开发人员支持众多服务器级芯片，并且在企业数据中心也已得到普遍认可。
