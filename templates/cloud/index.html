无论您是构建自己的云，还是想使用 公共云，Ubuntu 为您提供您需要的 所有软件基础设施、工具和服务。

使用 Ubuntu OpenStack 构建自己的云

Ubuntu OpenStack 是 Ubuntu 服务器最新版本和OpenStack 最新版本的组合版。OpenStack 主要发行版本获得 Ubuntu LTS 版五年支持，具有 OpenStack 最新功能，方便用户安心使用。

Ubuntu 拥有与云端整合的悠久历史，能比任何其他操作系统都更快速和更简单地进行部署。
