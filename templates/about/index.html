关于 Ubuntu

Ubuntu 是一款开源操作系统，可在个人电脑、智能手机、平板电脑、服务器和云上使用。这是全球社区基于 Canonical 策略和设计指南进行开发的结果；Canonical 是一家为配合 Ubuntu 开发而成立的软件公司。

Canonical 还为 Ubuntu 提供支持和服务，以扩大其全球市场覆盖率；这是因为推进创新的最佳方式就是为创新者提供他们所需要的技术。

Ubuntu 和 Canonical

Canonical 的服务确保 Ubuntu 通过认证，可在移动设备、个人电脑、服务器和云基础设施（公共或私人）上使用。世界各地的领先组织都对我们的服务和专业知识（包括系统管理以及在他们自己的云、服务器和桌面上部署 Ubuntu）青睐有加。
